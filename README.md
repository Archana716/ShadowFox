# ShadowFox Internship Tasks

## Task 1: Enhanced Console-Based Calculator
A Java console-based calculator built using BigDecimal for precision.

### Features
- Basic arithmetic operations
- Scientific calculations (square root, power)
- Temperature conversion
- Currency conversion
- Expression evaluation with BODMAS
- Exception handling for invalid input

### Technologies Used
- Java
- BigDecimal
- Scanner
- Shunting Yard Algorithm (Expression Parsing)

### How to Run
```bash
javac EnhancedCalculator.java
java EnhancedCalculator
```
