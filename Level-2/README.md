
# Task 2: Inventory Management System

A Java-based Inventory Management System developed as part of **Level-2 ShadowFox Internship Tasks**.  
This application helps manage products, stock levels, and basic inventory operations through a console-based interface.

---

## Features
- Add new products to inventory
- Update product quantity
- Remove products from inventory
- View all available products
- Search products by ID or name
- Input validation and exception handling

---

## Technologies Used
- Java
- Collections Framework (ArrayList / HashMap)
- Scanner
- Object-Oriented Programming (OOP)

---

## How to Run
```bash
javac InventoryBillingApp.java
java InventoryBillingApp
